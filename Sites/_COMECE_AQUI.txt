⚡ ATALHOS DO TECLADO
═══════════════════════════════════════════════════════════════

Ctrl + O  →  Abrir arquivo
Ctrl + S  →  Salvar arquivo
Ctrl + F  →  Procurar no arquivo
Ctrl + Z  →  Desfazer
Ctrl + F5 →  Atualizar navegador (limpa cache)
F12       →  Abrir Console (ver erros)
